## Introduction
ENEXT is a live platform CMS System for events, allowing the possiblity to create, edit, and delete events depending on your role as a user. Combined with a dashboard allowing profile display, statistics and more possibilites, you have many options that are desirable for a CMS system. 


## Built with a Laravel (PHP) Framework
- Frontend: Tailwind CSS, Alpine js
- Backend: Livewire with Volt (Class-based API style), MySQL database


## Feature 1: User role switch
Switch from user role to see the changes and limitations which is only implemented for the sake of this demo so you don't have to create your own accounts or login all the time. You're free of course to create an account if you like.

## Feature 2: Event viewing & favourite (users only)
View any visible event with an option to like it. This will be found in your dashboard favourites.

## Feature 3: Assign user roles (admins only)
Change a user role to [**admin**], [**organizer**] or [**user**] which will decide their privileges.
